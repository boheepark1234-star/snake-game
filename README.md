## How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/boheepark1234-star/snake-game.git
   cd snake-game
   ```
2. Open `index.html` in your web browser.

## Deploy with GitHub Pages
1. Go to your repository's **Settings > Pages**.
2. Set the source to the `main` branch and `/ (root)` directory.
3. Save and wait a few moments for the deployment link to appear.
4. Visit your GitHub Pages URL to play!
